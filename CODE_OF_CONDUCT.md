# Code of conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful, assume good faith, and keep discussion focused on the work.
Harassment, personal attacks, and discriminatory language are not tolerated in issues,
pull requests, discussions, or any other project space.

## Reporting

Report unacceptable behavior privately to the maintainers through GitHub — open a
[private security advisory](https://github.com/santhreal/veyyon/security/advisories) if
you need a confidential channel, or contact a maintainer directly. Reports are handled
discreetly. Maintainers may remove content, and block or ban accounts, that violate
this policy.

The full text of the Contributor Covenant 2.1, including the enforcement guidelines,
applies and is incorporated by reference.

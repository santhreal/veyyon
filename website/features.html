Features

What the binary includes

Agent loop and TUI: TypeScript on Bun. Grep, PTY/shell, hashline apply: Rust natives (@veyyon/natives). Operator detail: handbook under docs/.

Editing

Hashline and plan mode

edit / write use content-hash anchors; verification runs before disk write. Schema repair may rewrite tool JSON before validation. Plan mode drafts a plan file; resolve handles plan approval.

  • Hashline: anchors from prior reads; stale tags fail with recovery context.
  • Tool-call repair: parse leniency, alias rename, strict unknown keys at dispatch.
  • Plan / goal modes: engine modes; /plan, /goal.
  • Review: /review and optional advisor role.
# hashline edit path
edit routes/upload.ts
  anchor from read snapshot tag
  verify → write or error tool result
Control

Approvals (no OS sandbox)

tools.approvalMode gates tool tiers. There is no Landlock, seccomp, or Seatbelt confinement. Schema default is yolo (auto-approve all tiers).

Tiers

read / write / exec. Modes: plan, ask, auto-edit, yolo. Aliases: always-ask→ask, write→auto-edit.

Execpolicy

User and project .rules refine which commands auto-run. Untrusted project dirs disable project rules/hooks until trusted.

Non-interactive

veyyon --print (-p). No TTY for prompts; set --approval-mode or --yolo. Exit status follows the run.

Context

Compaction, memory, sessions

compaction.strategy: handoff | snap (default snap). Memory backends opt-in. Session trees: branch / fork / tree.

Memory

Optional backends store project facts across sessions. Disabled until enabled; content is inspectable on disk.

Session branching

Session files are trees. /tree jumps, labels, and branches; /fork and /branch write a new session file without rewriting the original.

Cockpit

TUI surface for concurrent subagents: per-agent model, thread focus, and live tool activity for the parent session.

Extend

Extensions surface

MCP servers, skills, TypeScript extensions, browser, and web search attach to the same agent loop. Non-interactive runs use veyyon --print.

MCP, skills, hooks

MCP servers register tools. SKILL.md files register procedures. Extensions register tools, commands, and pi.on lifecycle handlers.

Browser & web search

Built-in headless browser tool and multi-provider web search for live page content and search results.

Doctor & keybindings

veyyon plugin doctor checks install health. Keybindings: keybindings.yml under the active agent dir; vim mode is a setting.

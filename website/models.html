Models

Model selection

Provider catalog + credentials on the host. Interactive model, optional roles, and slot overrides. Handbook: Models, roles, and profiles.

Config

Interactive, roles, overrides

Interactive model is persisted as modelRoles.default (hidden role key). Selectable roles: smol, slow, vision, plan, designer, commit, tiny, task, advisor. Unset roles inherit interactive (except advisor’s own chain). subagent.model overrides modelRoles.task when set.

  • Interactive — /model, --model → modelRoles.default
  • Roles — modelRoles / settings → Model → Roles; flags --smol, --slow, --plan
  • Overrides — subagent.model, compaction.model
  • Cycle — cycleOrder default smol, slow
# ~/.veyyon/profiles/default/agent/config.yml
modelRoles:
  default: openai/gpt-5
  smol: openai/gpt-4.1-mini
subagent:
  model: deepseek/deepseek-chat
compaction:
  model: openai/gpt-5-mini

Roles

Named models in modelRoles (settings → Model → Roles). Key default stores the interactive model; it is hidden from role pickers. Custom role names allowed via config.

Compaction

compaction.strategy: handoff | snap (default snap). compaction.thresholdPercent, compaction.model.

Providers

Catalog and local discovery

Hosted providers in the bundled catalog; Ollama, LM Studio, and llama.cpp when the local server is running. BYOK keys go to the endpoint you configure.

# Getting help

- **Using veyyon** — start with the [handbook](docs/handbook/src/) (install,
  quickstart, features, troubleshooting, FAQ).
- **A bug or a feature request** — open a [GitHub issue](https://github.com/santhreal/veyyon/issues).
  Include your veyyon version (`veyyon --version`), OS, and steps to reproduce.
- **A question or an idea** — open a [Discussion](https://github.com/santhreal/veyyon/discussions).
- **A security vulnerability** — do **not** open a public issue. Follow
  [SECURITY.md](SECURITY.md).
- **Contributing** — see [CONTRIBUTING.md](CONTRIBUTING.md); note that pull requests
  require a vouch.
